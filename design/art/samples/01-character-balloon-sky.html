이 샘플에서 보여주는 것

  • R1 (Neon Glassblowing): 풍선 = 반투명 유리 + 내부 발광 + 외부 글로우 + rim highlight
  • R2 (Frosted Sky): 배경 그라데이션 #B5D8E8 → #F5C2D4 → #C4B0E0 (스카이블루 → 소프트핑크 → 라벤더)
  • R3 (Liquid Glass): UI 점수 패널 = backdrop-filter: blur(12px) saturate(140%) + 흰색 보더 + 미세 그림자
  • Critical 다크닝: 0.2초 동안 배경 → 딥 쿨블루로 전환, 풍선/캐릭터는 흰색-핫 글로우. P3 필러("운은 자주, 실력은 깊게")의 시청각 표현
  • 캐릭터 (humanoid glass figure): 작은 유리 인형. 머리·몸통·팔·다리. 작살 잡는 자세. drop-shadow 글로우로 떠다니는 듯한 느낌
  • 작살: 흰색-시안 그라데이션 + 화살촉. 풍선과 같은 유리 어휘
  • 콤보 인디케이터: P2 필러("화면이 점수보다 먼저 말한다") — "×8 CHAIN" 텍스트는 캐릭터 주변에 글로우 펄스

이 샘플에서 일부러 안 한 것 (디자인 결정)

  • 다크 배경 거부: Critical 순간만 다크닝 (Mock B). 95% 시간은 라이트 (사용자 결정)
  • 비비드·만화 톤 거부: 채도 낮춤, 파스텔 베이스 (Suika Game 톤과 의도적 차별화)
  • 픽셀·레트로 거부: 모든 요소가 부드러운 곡선·그라데이션 (Neon Glassblowing 정체성)
  • 장식 거부: 화면 가장자리 깔끔 (P2 필러 — 정보 우선)

기술 노트

  • Pixi.js v8 구현 시: 풍선은 Graphics + radial gradient (또는 sprite + @pixi/filter-glow)
  • 배경: 단일 Sprite with gradient texture, 또는 Mesh with vertex colors
  • UI: HTML overlay (CSS backdrop-filter 활용) 또는 Pixi Container + BlurFilter
  • Critical 다크닝: 배경 sprite의 tint 보간 (0.2초) + 전체 화면 white flash overlay (0.1초)
  • 캐릭터: SVG → Pixi sprite로 import, 또는 Graphics로 절차적 그리기
